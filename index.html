Resilience Patterns in Action

Chaos Monkey

Injects random failures and latency spikes to emulate unstable production dependencies.

Retries

Re-attempts transient failures before giving up, improving success in noisy network conditions.

Circuit Breaker

Uses CLOSED → OPEN → HALF_OPEN states to fail fast, recover safely, and avoid cascading outages.

Fallback Response

Returns degraded-but-usable responses when retries are exhausted or the breaker is open.
